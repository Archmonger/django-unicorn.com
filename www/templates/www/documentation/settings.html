Settings

django-unicorn stores all settings in a dictionary under the DJANGO_UNICORN setting attribute in the Django settings file. 

# settings.py
DJANGO_UNICORN = {
    "MINIMIZE": False
}

MINIMIZE

Provides a way to control if the minimized version of unicorn.min.js is used. Defaults to the negation of DEBUG.
