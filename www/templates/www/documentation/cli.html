CLI

django-unicorn provides a Django management command to easily create new components. 

python manage.py startunicorn hello-world

The command will create a unicorn directory, and templates and components sub-directories if necessary. Underneath the components directory there will be a new module and sub-class of the django_unicorn.components.Component class. Underneath the templates/unicorn directory will be a example template.

For the example above the folder structure would look like the following. 

unicorn/
    components/
        hello_world.py
    templates/
        unicorn/
            hello-world.html
