Related projects

django-unicorn stands on the shoulders of giants.

  • In particular Livewire, a fullstack framework for the Laravel web framework PHP (which django-unicorn borrows from heavily).
  • It is also inspired by LiveView, a package for the Elixir web framework, Phoenix.

django-unicorn would not be possible without a few packages.

  • Django, The web framework for perfectionists with deadlines.
  • Intelligent DOM merging with morphdom.
  • Landing page CSS from Bulma.
  • Documentation CSS from docsify.

There are a few Django packages which are tangentially-related and might be interesting.

  • Reactor, a LiveView library for Django.
  • django-page-components, a minimalistic framework for creating page components and using them in your Django views and templates.
