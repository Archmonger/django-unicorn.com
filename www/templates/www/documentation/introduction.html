Introduction

django-unicorn is a magical full-stack framework for Django. Instead of building additional API endpoints just to provide some interactivity, django-unicorn is a much simpler way to provide pieces of a modern website experience without moving away from all of the benefits of Django.

The goal is provide ways to seamlessly extend Django past its server-side framework roots without giving up all of its niceties or having to re-build a site in a frontend framework.

Related projects

django-unicorn stands on the shoulders of giants, in particular morphdom which is integral to making the fast DOM changes happen.

django-unicorn has also been heavily inspired by a few projects in other languages and frameworks.

  • Livewire, a full-stack framework for the PHP web framework, Laravel.
  • LiveView, a library for the Elixir web framework, Phoenix.

There are a few Django packages which are related to django-unicorn and might be interesting.

  • Reactor, a port of Elixir's LiveView to Django. Especially interesting for more complicated use-cases than django-unicorn like chat rooms, keeping multiple browsers in sync. Uses Django channels and websockets to work its magic.
  • django-page-components, a minimalistic framework for creating page components and using them in your Django views and templates.
