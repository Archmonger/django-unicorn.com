Add interactive elements easily

Don't wrestle with a frontend framework just for a few pieces of standard website functionality. Quickly add in simple interactions with standard Django templates and an intuitive server-side class.

Skip the separate Node.js watcher

Fighting with your Gulp/Grunt/Webpack configuration is a lot of fun, but might not be the best use of time. Skip dealing with the latest new frontend framework and use the Django skills you already know and love.

Don't build out an API if you don't need it

Django REST framework handles a lot of the hard parts of building a whole REST API interface, but it is still complicated and another framework to learn and understand. Skip it and use the powerful Django ORM just like you do for server-side applications.

Example Hello World component


{% verbatim %} 
<!-- hello-world.html -->
<div>
    <input unicorn:model="name" type="text" id="text">
    Hello {{ name|title }}
</div>
{% endverbatim %} {% verbatim %} 
# hello_world.py
from django_unicorn.components import UnicornView

class HelloWorldView(UnicornView):
    name = "World"
{% endverbatim %}

Then, include the component in a Django template.

{% verbatim %} 
<!-- index.html -->
{% unicorn 'hello-world' %}
{% endverbatim %}
